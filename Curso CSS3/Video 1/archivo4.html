Wild card

El selector wild car se unicia con un asterisco * e indica que el estilo se aplicara a todo el contenido de nuestra pagina web

tambien tenemos gerarquia

el cual funciona aplicando el estilo de la etiqueta mas genral a la mas espificada

por ejemplo el sistema elige una etiqueta o wild card para aplicar el estilo, seguido de una clase y seguido de un id

la regla "!important" aplicado en un css inidca que este estilo se aplicara sin importat gerarquias
